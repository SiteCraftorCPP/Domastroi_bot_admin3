# Backup PostgreSQL (Domastroi)

Файл `*.dump` снят утилитой `pg_dump -Fc`.

**Не публикуйте этот дамп.** В нём могут быть персональные данные пользователей бота и служебные таблицы. Держите репозиторий приватным и не пересылайте дамп в открытые каналы.

## Восстановление на новом сервере (пример)

```bash
# создать БД и пользователя заранее, затем:
pg_restore --no-owner --no-acl -h localhost -U domastroi -d domastroi -v backup/domastroi_YYYY-MM-DD_HHmm.dump
```

Если нужны владельцы/права как на старом сервере — уберите `--no-owner --no-acl` и восстанавливайте под суперпользователем Postgres.
